# PotHoles-Detection

## Create venv:
### On Linux
```bash
python3 -m venv venv
```
### On Windows
```bash
python -m venv venv
```

## Activate venv:
### On Linux
```bash
source .venv/bin/activate
```
### On Windows
```bash
.\.venv\Scripts\Activate
```

## Install python libraries
```bash
pip install -r requirements.txt
```
